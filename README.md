# Laravel 12 untuk Pemula: Project Pertama Anda

Selamat datang di tutorial Laravel 12 dalam bahasa Indonesia! Tutorial ini akan memandu Anda membuat project blog lengkap dari awal hingga siap deploy.

## 🎯 Tentang Tutorial Ini

Tutorial ini diadaptasi dan diterjemahkan ke bahasa Indonesia dari kursus populer Laravel untuk pemula. Kami menggunakan pendekatan **project-based learning** - Anda akan belajar sambil membangun aplikasi blog yang nyata.

### Yang Akan Anda Pelajari
- ✅ Instalasi dan setup Laravel 12
- ✅ Routing dan Blade templating
- ✅ Database migrations dan models
- ✅ Authentication dengan Laravel Breeze
- ✅ CRUD operations (Create, Read, Update, Delete)
- ✅ Form validation dan error handling
- ✅ Optimasi performance dasar

### Prasyarat
Sebelum memulai, pastikan Anda sudah memahami:
- **PHP dasar** (variables, functions, arrays, OOP)
- **HTML & CSS** dasar
- **Command line/Terminal** dasar

### Durasi Belajar
- **Total**: 15 pelajaran (~90 menit)
- **Format**: Text-based dengan code examples
- **Tingkat**: Pemula hingga menengah

## 🛠️ Persiapan Environment

Sebelum memulai tutorial, pastikan sistem Anda sudah terinstall:

```bash
# PHP 8.2 atau lebih tinggi
php --version

# Composer
composer --version

# Node.js dan NPM
node --version
npm --version

# Git
git --version
```
